Please follow this guide CAREFULLY to install BetterDiscord, if the regular installer will not work! Any OS will
work for this, but please go to your specific OS.

<Video client:load video="https://user-images.githubusercontent.com/90428263/162848023-9816afe8-8e77-44b5-afe2-054fd9131700.mp4" title="Manual% Speedrun in 2:28.63 by Disease" />

1\. Install <ExternalLink href="https://github.com/git-for-windows/git/releases/download/v2.34.1.windows.1/Git-2.34.1-64-bit.exe">Git</ExternalLink> and <ExternalLink href="https://nodejs.org/dist/v16.13.1/node-v16.13.1-x64.msi">Node.js</ExternalLink> **(Required)**

2\. Open up Windows Powershell (NOT Administrator) and check that you are NOT in the system32 directory. Change that with the follow command:

`cd ~/`

3\. Copy/paste the following below:

<Tabs client:load tabs={['Powershell', 'Command Prompt']}>
    <div class="choice active">
        <CopyInput
            replace="; " />
    </div>
    <div class="choice"><CopyInput /></div>
</Tabs>

These are the commands set out:

`git clone https://github.com/BetterDiscord/BetterDiscord.git `

`cd ./BetterDiscord/`

`pnpm install`

`pnpm run build`

`pnpm run inject`

For that last command, if you are using a non-stable install of Discord, add `canary` or
`ptb` at the end of the command.
