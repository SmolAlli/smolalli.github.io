SmolAlli's guide to using the vanilla command to get access to regular Discord temporarily.

To restore: Close down Discord completely, then re-open it.

### How to use the `--vanilla` command:

1\. Close Discord

2\. Press <kbd>âŠžWin</kbd> + <kbd>R</kbd>

3\. Paste `%localappdata%\Discord\app-1.0.9006\discord.exe --vanilla` into the chatbox that appears

### Vanilla Discord Desktop shortcut:

1\. On the desktop, right click, choose "New" and then "Create Shortcut"

2\. Copy and paste the following code into the filepath field, depending on what you want to have the shortcut do.

<Tabs client:load tabs={['Restart Discord with no BD', 'Open an already closed Discord with no BD']}>
    <div class="choice active">
        <CopyInput input="powershell.exe Stop-Process -Name Discord; start %USERPROFILE%/AppData/Local/Discord/app-1.0.9006/Discord.exe --vanilla" />
    </div>
    <div class="choice"><CopyInput input="powershell.exe start %USERPROFILE%/AppData/Local/Discord/app-1.0.9006/Discord.exe --vanilla" /></div>
</Tabs>

3\. Choose a name for the shortcut. You can also get the icon that Discord uses from `%USERPROFILE%/AppData/Local/Discord/app.ico`.
